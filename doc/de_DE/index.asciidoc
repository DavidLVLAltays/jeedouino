= Jeedouino v1.00

== Beschreibung
Dieser Plugin ermöglicht es Ihnen, die verschiedenen Arten von Ein-/Ausgänge der folgenden Karten zu kontrollieren: 

* Arduinos (verschiedene Modelle) angeschlossen an USB von Raspberry PI.
* Arduinos (différents modèles) branchés avec un shield Ethernet (W5100, W5500, ENC28J60) sur votre réseau.
* Der ESP8266 / NodeMCU / Wemos (wifi).
* Den GPIO vom Raspberry PI.
* Der PiFace-Karte am Raspberry PI.
* Der Karten, IO-Pi-Plus und MCP23017 auf Raspberry PI.


Il fonctionne en local ou en déporté sur votre réseau.

Il est compatible avec le fonctionnement maître/esclave (plugin Jeedom Link) de Jeedom.
